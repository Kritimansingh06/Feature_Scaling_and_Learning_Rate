<h1 align="center">📊 Feature Scaling and Learning Rate Demonstration</h1>

<p align="center">
This project demonstrates the importance of <b>feature scaling</b> and <b>learning rate selection</b> in Gradient Descent optimization for Linear Regression models.
</p>

---

<h2>🚀 Project Overview</h2>

<ul>
<li>Implements Linear Regression from scratch using NumPy</li>
<li>Custom Gradient Descent algorithm implementation</li>
<li>Shows effect of learning rate on convergence</li>
<li>Uses housing-style dataset features</li>
<li>Visualizes feature vs price relationships</li>
</ul>

---

<h2>🧠 Algorithms Implemented</h2>

<ul>

<li>Parameter Optimization</li>
</ul>

📊 Feature Scaling and Learning Rate Demonstration

This project demonstrates the importance of feature scaling and learning rate selection in Gradient Descent optimization for Linear Regression models.

🚀 Project Overview

  • Implements Linear Regression from scratch using NumPy
  • Implements custom Gradient Descent algorithm
  • Shows effect of learning rate on convergence
  • Uses real-style housing dataset features
  • Visualizes feature vs price relationships

📂 Dataset Features

FeatureDescription
SizeHouse size in square feet
BedroomsNumber of bedrooms
FloorsNumber of floors
AgeAge of house in years

🧠 Algorithms Implemented

  • Cost Function (Mean Squared Error)
  • Gradient Calculation
  • Batch Gradient Descent
  • Parameter Optimization

📈 Visualization

The notebook plots relationships between each feature and target price to help understand:

  • Feature influence
  • Scaling necessity
  • Training behavior

⚙️ Requirements

  • Python 3.x
  • NumPy
  • Matplotlib
  • Jupyter Notebook

▶️ How to Run

  1. Clone repository
  2. Open terminal in project folder
  3. Run Jupyter Notebook

📊 Feature Scaling and Learning Rate – ML Experiment

This project demonstrates the importance of feature scaling and learning rate selection in machine learning optimization using Gradient Descent.

🚀 Project Overview

Improper feature scaling and incorrect learning rate values can severely affect model training.

  • Slow convergence
  • Unstable optimization
  • Poor performance

This notebook explores:

  • Gradient Descent behavior
  • Impact of feature scaling
  • Learning rate tuning
  • Cost convergence visualization

📈 Notebook Demonstrations

1️⃣ Without Feature Scaling

  • Slow convergence
  • Oscillations in cost
  • Poor optimization

2️⃣ With Feature Scaling

  • Faster convergence
  • Stable cost decrease
  • Efficient updates

3️⃣ Learning Rate Experiments

  • Small learning rate → slow training
  • Optimal learning rate → fast convergence
  • Large learning rate → divergence
